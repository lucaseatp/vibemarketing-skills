# Search: keywords and copy

## Value-first angles (RSA)

For cold traffic or skeptical niches, prefer **value-led** headlines over empty promises. Teach or provoke with substance, then CTA.

| Value angle | RSA example (adapt to char limits) |
| --- | --- |
| List / tips | "5 Google Ads mistakes costing you leads" |
| Myth bust | "Low budget Search can work — if you fix this" |
| Specific how-to | "Exact match + negatives first — here's why" |
| Qualifier + tip | "For clinics: pre-qualify in the ad, not the form" |

Full framework (hooks, video scripts, repurposing): use the `copywriting` skill → `references/value-ad-creative.md`. Compress to RSA; do not paste long-form copy into headlines.

## Keywords

- **Transactional** themes (service + city, “near me”, quote, book).
- Start: **exact** `[term]` + **phrase** `"term"` — not broad on cold start.
- **Low** tier (&lt; US$ 200/day): favor lower CPC in Planner **if** intent and value still hold (cheap often = weak lead).
- Negatives from day 1: jobs, free, DIY, wrong product, competitor (if applicable).

## RSA — what moves results

Distinct angles in headlines (not 15 variations of the same CTA):

1. Intent (keyword + geo)
2. Benefit/outcome
3. Proof (only if confirmed)
4. Offer/CTA
5. Differentiator
6. Objection (speed, real guarantee, local)

**Differentiation:** copy only **you** can claim (proof, method, niche). Generic AI headlines saturate the auction.

**Ad strength:** cosmetic metric — do not optimize for it.

5 well-angled headlines &gt; 15 repeated headlines.

## Pre-qualification (low tier: &lt; US$ 200/day)

You only pay on click — use headlines to filter non-buyers:

- Price band (“from $X/month”)
- Profile (“ideal for 65+”, B2B)
- Tight geo (“South Zone only”)

**Pin** only when the line must show (price, compliance). CTR may drop; goal is post-click conversion.

## AI Max (advanced test, not launch)

- Turn on **text customization** (assets + site) — ok to test.
- Turn off **final URL expansion** by default (avoids wrong LP for services); ecommerce may test with exclusions (thank-you, checkout).
- Requires negatives + search-term review if using reach expansion.

## Extensions

Sitelinks (≥4 real URLs), callouts (≥4), snippets, call, location — only with true data.

**Google lead forms:** avoid at start (spam); prioritize on-site form with tracking.

## Landing page

- Hero = ad promise in 5s; one CTA; fast mobile.
- Privacy: clear purpose on forms (GDPR/LGPD as applicable).
