# Optimization checklist

## Weekly (new Search, weeks 1–6)

- [ ] Search terms → negatives + promote winners to exact
- [ ] “Limited by budget” on profitable campaigns?
- [ ] Lost IS: rank vs budget (interpret separately)
- [ ] Mobile vs desktop CPA
- [ ] Geo with spend and zero conversions → pause
- [ ] Weak RSAs after volume → replace; keep one variant testing
- [ ] **Low tier (&lt; US$ 200/day):** pause keywords with ROAS/CPA worse than average **before** full statistical significance

## PMax (if live)

- [ ] Leads by channel (Display vs Search vs YouTube) in reports
- [ ] CRM: % qualified leads vs other sources
- [ ] CPL vs **cost per qualified lead** (form with questions)
- [ ] Do not raise budget on volume alone

## Monthly

- [ ] Ads vs CRM/revenue
- [ ] Refresh creatives before fatigue
- [ ] tCPA/tROAS eligibility
- [ ] Reject generic “increase budget” recommendations without economics
- [ ] Policy disapprovals

## Diagnosis tree

```text
Low impressions → budget, bids, narrow keywords, disapproval
Impressions, low clicks → RSA, rank, irrelevant terms
Clicks, zero conversions → tag, LP, offer, form, intent
Expensive conversions → query mix, LP, bids, geo/device waste
Good CPA, want volume → +budget on winners; PMax only if CRM quality ok
```

## Common waste

- Broad match without negatives at start
- Display enabled on Search campaign
- Search partners not analyzed
- PMax remarketing &gt; acquisition without new-customer goals
- Duplicate keywords across campaigns
- Native lead form spam

## Scaling

Raise 15–20% every 5–7 days **if** for 2+ weeks: tracking ok, CPA/ROAS on target, on-theme terms, stable LP, (PMax) leads close in CRM.

## User report snippet

Spend · clicks · conversions · CPA/ROAS · top 5 converting terms · top 5 costly non-converters · 1 action with expected impact · (lead) CRM qualification rate if available
