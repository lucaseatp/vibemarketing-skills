# Post History Index

Use this file as a dynamic database connecting each group name, URL, and post history file.

Before suggesting new content:

1. Find the group in this index.
2. Open the file listed in `history_file`.
3. Read previous posts, results, comments, and learnings.
4. Suggest content based on the history, key people, group rules, and the user's current context.
5. If the group does not exist, create a file from `post-history/_post-history-template.md` and register the row in this index.
