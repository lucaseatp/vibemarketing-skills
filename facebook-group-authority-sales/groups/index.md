# Group Index

Use this file as a dynamic database connecting each group name, URL, and rules/context file.

Before creating content for a group:

1. Find the group in this index.
2. Open the file listed in `rules_file`.
3. Confirm rules, allowed CTAs, score, key people, and next allowed posting date.
4. If the group does not exist, create a file from `groups/_group-rules-template.md` and register the row in this index.
